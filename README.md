# TamperMonkey

[Download TamperMonkey Chrome Extension](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)

In Chrome Manage TamperMonkey Manage Extension, turn on Developer Mode and Allow User Scripts.


@match is to match the URLs where this script can run. In the Speed Kaltura script, the Kaltura video is embeded, so we also need to match the URL of where the video is embedded in.
